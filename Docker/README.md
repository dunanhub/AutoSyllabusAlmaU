# Docker

Заготовка для будущей контейнеризации проекта.

Планируемое назначение:

- Dockerfile для frontend;
- Dockerfile для backend;
- compose-файлы для локального запуска;
- переменные окружения и сетевые настройки.

Пока контейнеры не обязательны для запуска. Frontend запускается из папки `Frontend`.
